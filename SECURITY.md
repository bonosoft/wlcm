# Security Policy

## Supported Versions

The project will by supported with security updates as soon as it get to first public release.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please use the "Report a vulnerability" button on GitHub Security panel for reporting security related issues.

Use the Issues panel for all non-security related bugs and other feature requests.
